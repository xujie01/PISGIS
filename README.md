# GIS数字平台

## 项目介绍

基于WebAppBuilder框架修改的noportal二维GIS平台。

## 功能介绍

包含基本的GIS功能、统计图、标绘编辑、地图导航、全景地图、轨迹回放等。


### 技术选型

**前端技术**:

* ArcGIS API for JavaScript
* WebAppBuilder
* jquery
* echart
* Layer
* threejs

## 环境搭建

### 开发工具:


* SVN: 版本管理

* Webstorm: 开发IDE
